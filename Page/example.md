# Markdown 编写指南

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。本博客系统完美支持 Markdown 渲染。

## 基础语法演示

### 1. 代码块

你可以轻松地插入代码块，并享受自动高亮：

```javascript
// 这是一个 JavaScript 示例
function sayHello() {
    console.log("Hello, RyuChan!");
}

sayHello();
```
